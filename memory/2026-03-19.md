## 2026-03-19

- `components/finance/people-roster-browser.tsx`: the roster filter card now uses a collapsed-by-default `Collapsible`; seat maintenance moved from the old budget-only override dialog into a broader seat editor dialog with manual seat creation from the roster page.
- `components/finance/seat-editor-dialog.tsx` and `lib/finance/queries.ts`: manual seats are created as `TrackerSeat` rows with `sourceType = MANUAL` and `seatId` assigned as the next available numeric integer across tracker seats and approved roster seat ids for the selected year.
- `prisma/schema.prisma`: seat-editable roster overrides now include `team`, `inSeat`, `description`, `band`, `location`, `vendor`, `manager`, and `dailyRate`; `TrackerSeat.manager` was added so manual seats and derived roster seats can both surface manager values.
- `prisma/schema.prisma` and `lib/finance/queries.ts`: allowed vendor/location/manager dropdown values are stored in a new year-scoped `SeatReferenceValue` table; page loads backfill this table from active seats and approved roster rows so the current dev data becomes the initial admin-maintained option set automatically.
- `components/finance/admin-browser.tsx` and `app/api/admin/seat-reference-values/[type]/route.ts`: Admin now has separate vendor, location, and manager maintenance cards with single-value CRUD plus CSV export/import using the same admin patterns as other backups/imports.
- `components/finance/roster-imports-browser.tsx` and `app/people-roster/imports/page.tsx`: roster import/upload and import-history management now live on a dedicated `/people-roster/imports` subpage with a back link to the main roster page; the main roster page keeps a `Manage Imports` button instead of embedding import cards inline.
- `components/finance/people-roster-browser.tsx`: the filter card is now visually ordered above the summary stat cards by using flex item ordering on the page sections instead of duplicating the filter markup.
- Validation for this change: `npm run db:generate`, `npm run db:migrate -- --name add_seat_reference_values_and_seat_editor`, and `npm run build`.
